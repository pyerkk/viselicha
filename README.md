# 🐶 Hangman: Pug Edition (^_^)v

Консольная игра **"Виселица"** на C++.

## ✨ Особенности проекта
*   📦 **Standalone EXE**: Все библиотеки (`libstdc++`, `libgcc`) вшиты внутрь через `-static`. Запускается без лишних DLL.
*   🖼️ **Custom Icon**: Вшитая в ресурсы иконка мопса для исполняемого файла.
*   🧩 **Modular Design**: Код разделен на заголовочный файл и модули реализации (`.h`, `.cpp`).
*   🧠 **2-Step Hints**: Система подсказок (1. Категория слова -> 2. Случайная буква).
*   📚 **Huge Library**: Более 350 слов на английском языке с тематическими описаниями.

## 📂 Структура файлов
*   `hangman.exe` — готовое приложение.
*   `main.cpp` — основной игровой цикл.
*   `functions.cpp` — логика отрисовки и база слов.
*   `hangman.h` — объявления структур и функций.
*   `Pug.ico` — картинка.
*   `res.rc` — скрипт ресурсов для иконки.
*   `res.res` — скомпилированный файл картинки.

## 🎮 Как играть
1. Запустите **`hangman.exe`**.
2. Вводите буквы на английской раскладке и нажимайте **Enter**.
3. Если застряли, введите **`!`**:
    *   **Первое нажатие**: Показывает категорию слова (например, *IT & Programming*).
    *   **Второе нажатие**: Открывает одну случайную букву.
4. У вас есть **6 попыток**, чтобы спасти человечка от виселицы!

## 🛠️ Сборка (MinGW/GCC)
Если вы хотите пересобрать проект самостоятельно:
1. Подготовьте файл `Pug.ico` в корневой папке.
2. Подготовь текстовый файл `res.rc` в корневой папке с текстом:
   ```bash
   MAINICON ICON "Pug.ico"
3. Скомпилируйте файл ресурсов:
   ```bash
   windres res.rc -O coff -o res.res
4. Скомпилируйте проект:
   ```bash
   g++ -static main.cpp functions.cpp res.res -o hangman.exe -static-libgcc -static-libstdc++
